# Understanding JavaScript Variables

## 1. What Are JavaScript Variables?

A variable is something used to store data in JavaScript.

I think of a variable like a container that keeps information so we can use it later in our code.

### Real Life Example

If I have a box with my books inside, the box stores the books. A variable does something similar because it stores information.

Example:


let name = "Eugene";


In this example, the variable `name` stores the value `"Eugene"`.

## 2. How Do You Declare Variables in JavaScript?

There are three ways to declare variables in JavaScript.

### let

`let` is used when the value might change later.

Example:

let age = 20;


### const

`const` is used when the value should stay the same.

Example:

const country = "Kenya";

### var

`var` is an older way of creating variables.

Example:

var school = "Moringa";


## 3. Rules for Naming Variables

Some rules for naming variables are:

- A variable name can contain letters, numbers, `_` and `$`.
- A variable name cannot start with a number.
- Spaces are not allowed.
- JavaScript keywords should not be used as variable names.
- Variable names are case-sensitive.

### Valid Variable Names

studentName
age1
_total

### Invalid Variable Names

1student
student name
const

Reasons:

- `1student` starts with a number.
- `student name` has a space.
- `const` is a JavaScript keyword.

## 4. Variable Naming Styles

### Camel Case

In camel case, the first word starts with a small letter and the next words start with capital letters.

Examples:

studentName
firstName

This style is commonly used for variables in JavaScript.

### Pascal Case

In Pascal Case, every word starts with a capital letter.

Examples:

StudentName
FirstName

This style is often used for classes.

### Snake Case

In Snake Case, words are separated using underscores.

Examples:

student_name
first_name

This style is common in databases and some programming languages.

## 5. Summary

In this exercise I learned that variables are used to store data in JavaScript. I also learned that variables can be created using `let`, `const`, and `var`. Variable names have rules that need to be followed, and there are different naming styles such as Camel Case, Pascal Case, and Snake Case.
